상단 2개 버튼(스니퍼, 서비스) 활성화 이후 게임 실행
※ 스니퍼를 끄면 칸코레에서 데이터를 받아오지 않음

주요 기능
- 원정 및 입거, 피로도 회복 알림
- 대파 알림 (출격 중 및 출격 전)
- 색적/제공치/피로도 표시
- 출격 중 현재 위치 표시
- 전투 예보 / 임무 번역
- 함선/장비 리스트, 경험치 계산
- 개수공창표, 원정조건표
- OpenDB/poi-statistics에 데이터 전송

업데이트 확인 시 계속 오류가 뜨는 경우에는 설정에서 업데이트 확인 서버를 [NOVA Server]로 변경하세요.
